# "Smart"christmas town
A ESP32 C6 based Christmas town which connects to homeautomation to be able to remotly turn on and off the christmas lights

## Features

* **Microcontroller:** ESP32-C6-Zero
* **Power Input:** 12V DC
* **supported** it works with 3 3V, 2 4.5V and one 12V houses/street lights but you can add more or use less based on youre christmas town
* **Connectivity:** USB-C receptacle for easy firmware flashing.
